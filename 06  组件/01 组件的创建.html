全局组件
全局组件一经创建就可以被所有Vue实例重复调用，所以仅用于复数使用的场景
语法：Vue.component('name', {...})
局部组件
局部组件被定义在一个Vue实例中，所以它的作用域仅在该实例
语法：new Vue({ components:{ 'name': { template:'...' } } })
使用 Vue.extend 创建组件
使用 Vue.extend 创建一个全局组件：
var com = Vue.extend({ template: '< h5>这是使用 Vue.extend 创建的组件< /h5>' })
Vue.component('name', com)

上面的代码也可以合并：

Vue.component('name', Vue.extend({ template: '< h5>这是使用 Vue.extend 创建的组件< /h5>' }))
常用的创建方式
通过在 HTML 中定义 template 元素来完成布置（注意：必须放在 Vue 实例控制区外部，即不在 el 内）
            < template >
                < div class="box" >
                    < li >...< /li >
                    < a href="" >...< /a >
                < /div >
            < /template >

注意：

  • template 是组件要展示的 HTML 结构，并且只能有一个根元素。也就是说比如< li >...< /li >< a >...< /a >会报错，需要用一个元素包裹它们，比如< div >< li >...< /li >< a >...< /a >< /div >
  • name 是组件的名称，要调用组件只需要在页面用组件名写一个标签
