命名路由---name属性标记路由
给路由定义不同的名字，根据名字进行匹配，同时，给不同 < router-link > 通过对应名称进行渲染。这需要我们在Router实例的 routes 中添加一个 name 属性
  • 此时，在 < router-link >中，我们使用 v-bind: 绑定 to 属性（简写 :to ），并且用一个对象承载，里面必须有路由地址，即 name 对应的路由
  • 如果地址有传参，使用 params 对象接收参数
  • 在Router实例中，配置的路由要使用 name: '名称' 匹配路由，名称要与 < router-link > 的相对应
    • 用户 登录 注册
